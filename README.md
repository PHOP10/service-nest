# nestjs-prisma
template backend with nestjs

cd to services [service/core]

Install Package

``` npm install ```

Prisma Setup

``` npm run generate ```

Run Dev

```npm run dev ```

Change Model

```npm run db:generate:clean```

Push DB

```db:push```
